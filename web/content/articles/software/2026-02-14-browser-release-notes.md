A major browser release this week shipped improved controls for memory allocation, background tab throttling, and extension isolation.

Administrators can now enforce stricter permission boundaries via policy templates, reducing risk from overly broad extension access.

For power users, the update also includes clearer diagnostics in the performance panel.
